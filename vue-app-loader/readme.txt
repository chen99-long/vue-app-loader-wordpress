通过短代码在WordPress中加载远程Vue应用。

== Description ==

Vue App Loader 插件允许你通过简单的短代码在WordPress页面或文章中加载远程Vue应用。

主要特点:
* 简单易用的短代码
* 支持远程CDN加载
* 自动处理CSS和JavaScript依赖

== Installation ==

1. 上传 `vue-app-loader` 文件夹到 `/wp-content/plugins/` 目录
2. 在WordPress后台启用插件
3. 在文章或页面中使用短代码: [vue_app cdn="你的CDN地址"]

== Frequently Asked Questions ==

= 如何使用这个插件? =

只需在文章或页面中插入短代码 [vue_app cdn="https://your-cdn-url/"] 即可。
